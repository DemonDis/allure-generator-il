## Tech Stack

- allure 2.20.1
- Node.js 64 bit 14.17.5
- Node.js 64 bit v14.21.3
- npm 10.2.3
- yarn 1.22.21


## Start Allure

```bash
$ allure generate logs --clean -o report
```
```bash
$ allure open -p 9999 report&
```
```bash
$ cp -r ./report/history ./logs
```
